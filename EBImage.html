3 Memanggil Package

Setelah menginstal, cobalah untuk mengecek apakah package benar-benar terinstal.

Dengan cara seperti ini:

library(EBImage)

4 Memanggil Gambar

Pada langkah ini, kita akan memanggil gambar yang akan di analisa menggunakan EBImage. Package (“EBImage”) saat ini mendukung tiga format file gambar: jpeg, png dan tiff. Dengan cara seperti ini:

setwd("C:/Agung/kuliah/Semester 2/Linear Algebra/EBImage2")
library(EBImage)
Image <- readImage('mosque.jpg')
display(Image)

5 Histogram

Untuk menampilkan histogram adalah sebagai berikut:

uwow <- readImage('mosque.jpg')
hist(uwow)

6 Method Browser

Pada penggunaan method browser digunakan untuk melakukan zoom out dan zoom in pada gambar.

Dan jangan lupa untuk menaruh file gambar pada directory project anda.

img = readImage("mosque.jpg")
display(img, method="browser")

7 Memberi teks pada gambar

Kalian dapat menambahkan teks pada gambar kalian dengan cara seperti berikut:

untuk keterangan x dan y adalah posisi teks yang ingin kalian atur.

display(img)
text(x = 300, y = 280, label = "Sholatlah kawan", adj = c(0,1), col = "white", cex = 2)

8 Mengatur Brightness

Mengatur kecerahan dapat dilakukan dengan cara mengatur pada simbol +/- pada r markdown.

setwd("C:/Agung/kuliah/Semester 2/Linear Algebra/EBImage2")
library(EBImage)
Image1 <- Image + 1.2
Image2 <- Image - 0.2
par(mfrow= c(1,2))
plot(Image1)
plot(Image2)

9 Mengatur Kontras

Kalian dapat mengatur gambar agar bisa diubah ketebalan kontrasnya. Pada opsi ini menggunakan simbol sama dengan (=) yang secara default gambar akan bernilai 1.

setwd("C:/Agung/kuliah/Semester 2/Linear Algebra/EBImage2")
library(EBImage)
Image3 <- Image * 0.5
Image4 <- Image * 2
par(mfrow= c(1,2))
plot(Image3)
plot(Image4)

10 Gamma Correction

Kalian dapat melihat gamma pada sebuah gambar dan mengatur gamma tersebut. Caranya mirip seperti mengatur kontras namun menggunakan simbol “^”.

setwd("C:/Agung/kuliah/Semester 2/Linear Algebra/EBImage2")
library(EBImage)
Image5 <- Image ^ 2
Image6 <- Image ^ 0.7
par(mfrow= c(1,2))
plot(Image5)
plot(Image6)

11 Meng-Crop Gambar

Kalian dapat memotong (Crop) gambar dengan cara sebagai berikut:

setwd("C:/Agung/kuliah/Semester 2/Linear Algebra/EBImage2")
library(EBImage)
display(Image[80:220, 1:200,])

12 Rotasi Gambar

Kalian dapat merotasi gambar menjadi miring. Disini saya akan mencontohkan sesuai value yang ada di Imagetr <- translate(rotate(Image, 45), c(50, 0))

setwd("C:/Agung/kuliah/Semester 2/Linear Algebra/EBImage2")
library(EBImage)
Imagetr <- translate(rotate(Image, 45), c(50, 0))
display(Imagetr)

14 Memberikan Filter

Pada proses ini gambar akan diberikan beberapa filter dan bisa dibilang proses untuk menjalankan program ini terbilang berat dan panjang.

Untuk memberikan filter adalah sebagai berikut:

setwd("C:/Agung/kuliah/Semester 2/Linear Algebra/EBImage2")
library(EBImage)
fLow <- makeBrush(21, shape= 'disc', step=FALSE)^2
fLow <- fLow/sum(fLow)
Image.fLow <- filter2(Image, fLow)
display(Image.fLow)

fHigh <- matrix(1, nc = 3, nr = 3)
fHigh[2, 2] <- -8
Image.fHigh <- filter2(Image, fHigh)
display(Image.fHigh)

aa<- medianFilter(Image.fHigh,1.1)
par(mfrow= c(1,2))
plot(aa)
plot(Image.fHigh)

Image <- readImage("mosque.jpg")
medFltr <- medianFilter(Image, 1.1)
display(medFltr)
